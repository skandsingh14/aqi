EcoQ India

Advanced Air Quality Monitoring & Prediction System

Project Description

The India Air Quality Prediction System is a full-stack data-driven web application designed to monitor, analyze, and predict the air quality of major Indian cities in real-time. It provides interactive visualizations, comparisons, and machine learning-powered predictions to help citizens make informed health decisions.

Technologies Used

  • Frontend: HTML5, CSS3 (Glassmorphism UI), Vanilla JavaScript
  • Libraries: Chart.js (Data Visualization), Leaflet.js (Interactive Maps)
  • Backend: Python, Flask Framework
  • Machine Learning: scikit-learn (Random Forest Regressor), pandas, numpy
  • Live Data: OpenWeatherMap Air Pollution API & Geocoding API

Machine Learning Model

The system employs a Random Forest Regressor algorithm to predict Air Quality Index based on real-time specific pollutant concentrations.

Features: PM2.5, PM10, NO2, SO2, CO, O3
Target: AQI Category Label

Random Forest was chosen for its robustness to outliers and non-linear relationships characteristic of environmental data.

Dataset & Operation

Training Data: A synthetically generated but statistically valid proxy dataset mirroring realistic pollution levels across Indian cities was used to train the base model.

Live Operation: In real-time, the application fetches the latest local components (PM2.5, etc.) from the OpenWeather API, feeds them into our pre-trained model.pkl, and outputs an accurately estimated and color-coded AQI.
